# News-GUI

A GUI that shows top 5 results of a query to NewsAPI.org

| Dependencies                                          |             |
| ----------------------------------------------------- | ----------- |
| [JavaFX 11.0.2](https://gluonhq.com/products/javafx/) | gui         |
| [gson 2.8.5](https://github.com/google/gson)          | JSON parser |
